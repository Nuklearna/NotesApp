# ToDo/Notes App

Simple front-end React Native app.

## Features

* Header
    * Just a title

* Footer
    * Text input for notes

* Button
    * Add notes

* Delete button
    * Delete note

* Notes
    * Date created
    * Text

* Styling
    * Custom CSS

## Built With

* [React Native](https://facebook.github.io/react-native/)
